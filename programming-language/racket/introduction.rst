安裝
========================================

正常版：

.. code-block:: sh

    $ sudo pacman -S racket
    ...
    Packages (1) racket-6.6-1

    Total Download Size:    43.31 MiB
    Total Installed Size:  227.15 MiB
    ...


瘦身版：

.. code-block:: sh

    # without DrRacket
    $ sudo pacman -S racket-minimal
    ...

    Total Download Size:    5.21 MiB
    Total Installed Size:  22.80 MiB
    ...
