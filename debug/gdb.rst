gdbgui - browser-based frontend for GDB
========================================

:Repo: https://github.com/cs01/gdbgui

把 GDB 的功能和資訊整合到網頁上，
用起來比較方便，
對於入門也比較友善。

安裝：

.. code-block:: sh

    $ sudo pacman -S gdb
    $ pip install gdbgui
