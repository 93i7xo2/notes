Transfer Learning
========================================


.. contents:: 目錄


介紹
========================================

Transfer Learning 指的是利用之前已經針對類似但不同的問題訓練過的 model 作為起始 model，
拿來針對自己要處理的問題做特化訓練，
藉此來加入得到好結果的效率，
例如利用本來訓練好分辨汽車的 model 來訓練分辨卡車用的 model。
