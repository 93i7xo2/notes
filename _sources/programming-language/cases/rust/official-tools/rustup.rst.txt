.. contents:: 目錄


介紹
========================================

:Site: https://rustup.rs/
:Repo: https://github.com/rust-lang/rustup.rs


rustup 是管理數個 Rust 版本的工具，
可以根據所在的專案來選擇要使用的 Rust compiler 版本。
（以前是使用叫做 multirust 的工具，後來被整個用 Rust 重新設計寫過）
