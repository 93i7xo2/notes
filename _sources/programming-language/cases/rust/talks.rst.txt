RustConf 2017 - Opening Keynote: Make Productivity a Core Value of Rust
=======================================================================

:Video: https://www.youtube.com/watch?v=COrl851gMTY


Rust 團隊的發展更新，
介紹當初 2017 的規劃，
目前 impl period 的狀況、
語言發展（NLL、impl Trait、const generic、async 等等）、
Libz Blitz 的目標和狀況，
對於 Rust 近期發展和未來走向有很好的說明。
