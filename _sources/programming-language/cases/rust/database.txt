r2d2
========================================

`r2d2 <https://github.com/sfackler/r2d2>`_
是 Rust 的一個資料庫連線 Pool 的 Library，
藉此可以避免在每次操作時重複建立連線。



Diesel ORM
========================================

`Diesel <https://github.com/diesel-rs/diesel>`_ 是 Rust 的一個 ORM Framework，
提供方便的 derive 可以幫 struct 產生需要的 SQL Query。
