syntect
========================================

:Repo: https://github.com/trishume/syntect


syntect 是一套 syntax highlighting 的 library，
借用了 Sublime Text 的語法定義檔案，
藉此來處理各個程式語言並進行上色，
可以選擇輸出顏色在 Terminal 或是轉成 HTML。

（可以想像 Python 生態圈的 Pygments）


相關：

* `Syntect Server <https://github.com/sourcegraph/syntect_server>`_
