* `Powerline <https://github.com/powerline/fonts>`_
* `Nerd Fonts <https://github.com/ryanoasis/nerd-fonts>`_
    - 蒐集多個 glyphs (icons) 專案，包含例如 Powerline、Font Awesome 等等
    - 整合起來為各個字型上 patches，專案也提供工具可以自己為想要的字型上 patches。



相關專案
========================================

* `HarfBuzz <https://github.com/harfbuzz/harfbuzz>`_
    - OpenType text shaping engine
    - 2.0
        + AAT shaping support
* `RustType <https://github.com/redox-os/rusttype>`_
    - Rust
    - pure Rust alternative to libraries like FreeType
* `Pathfinder <https://github.com/pcwalton/pathfinder>`_
    - Rust
    - GPU rasterizer for fonts and vector graphics
* `font-kit <https://github.com/pcwalton/font-kit>`_
    - Rust
    - 跨平台字型讀取、搜尋套件
    - 支援 Fontconfig、Core Text、DirectWrite 等等
