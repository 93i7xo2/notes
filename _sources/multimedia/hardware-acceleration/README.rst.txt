========================================
硬體加速 (Hardware Acceleration)
========================================

* `介紹 <introduction.rst>`_
* 硬體加速 API
    - 2002 - Video4Linux
    - 2004 - OpenMAX
    - 2007 - VA-API (Intel)
    - 2007 - OVD (AMD)
    - 2009 - VDPAU (NVIDIA)
    - 2010 - DCE (TI)
    - 2012 - NVENC/NVDEC (NVIDIA)
    - 2012 - Android MediaCodec (based on OpenMAX)
    - Apple VideoTool Box Framework

* 軟體支援
    - `GStreamer <gstreamer-hardware-acceleration.rst>`_
    - FFmpeg
    - Firefox
    - Chromium
    - VLC
