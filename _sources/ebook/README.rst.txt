========================================
E-Book
========================================

* Mobipocket (Mobi) (.mobi)
    - 法國公司 Mobipocket (成立於 2000 年) 建立的格式
    - 此公司於 2005 年被 Amazon 收購
* EPUB (.epub)
    - 由 IDPF (International Digital Publishing Forum) 制定的格式，於 2007 年釋出
* Kindle File Format (.azw, .azw3, .kfx)
    - AZW3 = KF8 = Kindle Format 8
    - KFX = Kindle Format 10
* DjVu
* `Comparison of e-book formats <https://en.wikipedia.org/wiki/Comparison_of_e-book_formats>`_
