========================================
書籍紀錄
========================================

書籍筆記以 ISBN 為開頭、書名接在後頭作為檔案名稱，
例如「 ``ISBN-13|978-1-59327-385-9|A Bug Hunter's Diary.rst`` 」。

.. code-block:: python

    "ISBN-{}|{}|{}.rst".format(13, "978-1-59327-385-9", "A Bug Hunter's Diary")
