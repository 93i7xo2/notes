fwupdate 使用
========================================

.. code-block:: sh

    $ fwupdmgr get-devices
    $ sudo fwupdmgr hwids
    $ sudo fwupdate -e
    Firmware updates will be enabled after the system is rebooted.
    $ sudo fwupdate -l
    $ sudo fwupdmgr update -v
